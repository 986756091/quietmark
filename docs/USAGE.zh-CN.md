# QuietMark 使用指南

[English](USAGE.md) | [简体中文](USAGE.zh-CN.md)

## 打开文件

可以使用以下任意方式：

- 点击工具栏中的文件夹按钮。
- 选择 `文件 -> 打开...`。
- 将 `.md`、`.markdown`、`.mdown` 或 `.txt` 文件拖到 macOS App 图标上。

打开 Markdown 文件后，QuietMark 会扫描同一文件夹，并在文件面板中展示附近的 Markdown 文件。

## 语言

点击顶部工具栏的语言按钮，可以在英文和简体中文之间切换。该设置会保存在本地，并同步更新工具栏文字、侧边栏标签、状态栏、编辑器占位文字、Vditor 内置语言和 macOS 菜单。

## 切换文件

文件面板会展示当前文件夹中的 Markdown 文件。点击文件名即可打开。

如果当前文件存在未保存修改，并且已经保存在磁盘上，QuietMark 会在切换前自动保存。如果当前文档仍是未保存草稿，请先保存，再切换到侧边栏中的其他文件。

## 大纲

大纲面板会列出当前文档中的标题。可以使用搜索框筛选标题。点击标题后，应用会切换到源码模式并跳转到对应位置。

对于很长的文档，大纲面板会独立滚动，不会和文件列表互相挤占。

## 折叠面板

文件和大纲面板标题左侧都有折叠按钮：

- 大纲很长时，可以折叠文件面板，为大纲留出更多空间。
- 文件夹中文件很多时，可以折叠大纲面板，为文件列表留出更多空间。
- 点击文件面板标题右侧的 `x` 按钮，可以关闭整个侧边栏。
- 点击编辑器工具栏中的侧边栏按钮，可以重新打开侧边栏。

## 编辑模式

- 即时：写作时实时查看 Markdown 渲染反馈。
- 富文本：所见即所得风格编辑。
- 源码：直接编辑 Markdown 原文。
- 阅读：以渲染后的阅读视图查看文档。

## 保存

- `文件 -> 保存` 或工具栏保存按钮会保存当前文件。
- `文件 -> 另存为...` 会将当前文档写入新文件。
- 状态栏会显示当前文档是否已保存。

## 导出 HTML

使用 `文件 -> 导出 HTML...` 或工具栏导出按钮，可以将渲染后的文档导出为独立 HTML 文件。

## 主题和专注

使用主题按钮在纸张和深色主题之间切换。使用专注按钮可以隐藏侧边栏并减少周围界面干扰。
