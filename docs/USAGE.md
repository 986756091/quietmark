# QuietMark Usage Guide

## Opening Files

Use any of these options:

- Click the folder button in the toolbar.
- Choose `File -> Open...`.
- Drag a `.md`, `.markdown`, `.mdown`, or `.txt` file onto the macOS app icon.

When a Markdown file is open, QuietMark scans the same folder and shows nearby Markdown files in the Files panel.

## Switching Files

The Files panel shows Markdown files from the current folder. Click a file name to open it.

If the current file has unsaved changes and already exists on disk, QuietMark saves it before switching. If the current document is still an unsaved draft, save it first before switching to another sidebar file.

## Outline

The Outline panel lists headings from the current document. Use the search box to filter headings. Clicking a heading switches to source mode and jumps to that heading.

For large documents, the Outline panel scrolls independently from the Files panel.

## Collapsing Panels

The Files and Outline panel headers both have a disclosure button:

- Collapse Files when you want more room for a long outline.
- Collapse Outline when you want more room for a large folder.
- Close the whole sidebar with the `x` button in the Files header.
- Reopen the sidebar with the sidebar button in the editor toolbar.

## Editor Modes

- Instant: live Markdown rendering while writing.
- Rich: WYSIWYG-style editing.
- Source: plain Markdown source editing.
- Read: rendered reading view.

## Saving

- `File -> Save` or the save toolbar button saves the current file.
- `File -> Save As...` writes the current document to a new file.
- The status bar shows whether the current document is saved or unsaved.

## Exporting HTML

Use `File -> Export HTML...` or the export toolbar button to write the rendered document as standalone HTML.

## Themes And Focus

Use the theme button to switch between paper and dark themes. Use the focus button to hide the sidebar and reduce surrounding UI.
